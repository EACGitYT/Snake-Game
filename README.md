# Snake Game

## Overview
This is a Snake Game developed in Python using Pygame.
I followed a tutorial to understand the fundamentals and then modified the code by adding my own improvements

## Features
- Snake movement
- Food generation
- Score counter
- Soundtracks
- Custom graphics

## Technologies
Python and Pygame (import pygame)

## What I learned:
- Game loops
- Event handling
- Collision detection

## Future Improvements
- Main menu
- High score system
